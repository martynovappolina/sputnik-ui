## Page - Страница
Является компонентом-оберткой для остальных компонентов.

### Пример использования
```
<Page>
  <Menu />
  <Cover />
  <Divider />
  <HorizontalCardsBlock />
  <VerticalCardWithImageAndText theme="dark" />
  <NumberedList />
  <TextBlockWithHeader theme="dark" />
  <MultiColumnListWithIcons />
  <Stages theme="dark" />
  <GridWithCards />
  <Contacts />
  <Footer />
</Page>
```
